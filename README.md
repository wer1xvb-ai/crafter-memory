# Crafter Memory

A Fabric mod that adds recipe memory functionality to vanilla Crafters in Minecraft 1.21+.

## Features

- 📖 **Recipe Book UI** - Browse and select recipes just like in a crafting table
- 💾 **Save Recipes** - Remember your favorite crafting recipes
- 🔄 **Replace/Forget** - Easily swap or delete saved recipes
- 🔌 **Hopper Integration** - Hoppers automatically feed ingredients in the correct order
- 👻 **Preview Icons** - Semi-transparent ingredient icons show what's needed

## Installation

1. Download the mod JAR from releases
2. Place it in your `mods` folder
3. Requires Fabric Loader and Fabric API

## Usage

1. Open a Crafter block
2. Click the recipe book button (like in a crafting table)
3. Select a recipe and click "Remember Recipe"
4. Hoppers will automatically load ingredients in the correct order
5. Click "Forget Recipe" to delete the saved recipe

## Requirements

- Minecraft 1.21+
- Fabric Loader
- Fabric API
